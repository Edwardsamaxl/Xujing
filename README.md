# 叙境 · 故宫数字文旅导览

> 不是千篇一律的语音讲解，而是一场为你量身定制的紫禁城探秘。

**叙境**是一款面向故宫等文化景区的智能叙事导览系统。游客扫码进入后，选择兴趣标签，AI 便会根据你的偏好、实时位置和剧情线索，为每一处冷门秘境生成独一无二的探索任务与故事。历史的每一块青砖，都在等待精确的解读。

---

## 核心亮点

### 1. AI 个性化叙事生成
每处景点不是背诵固定讲解词，而是根据你的兴趣标签（历史、建筑、悬疑、人物、亲子、工艺）实时生成专属剧情。同一座宫殿，悬疑爱好者听到的是未解之谜，建筑爱好者看到的是营造法式，历史爱好者则走进时光纵深。

### 2. 四因子智能推荐算法（v0.3）
系统不仅听你「说什么」，更看你「做什么」：
- **距离因子**：物理可达性优先，步行 15 分钟以内渐进引导
- **兴趣因子**：动态兴趣画像，随探索不断校准你的真实偏好
- **剧情连线因子**：叙事连贯性优先，点位之间有故事勾连
- **停留时间因子**（新增）：你在某类叙事上的真实停留，是最诚实的兴趣表达

### 3. 语音交互，解放双手
- **语音提问**：对着古迹直接发问，AI 基于景区 RAG 知识库作答，禁止编造无出处事实
- **语音朗读**：剧情与回答均支持 AI 语音合成，边走边听
- **实时字幕**：语音内容同步转文字，兼顾嘈杂环境与听障友好

### 4. 国风沉浸 UI
宣纸纹理底色、水墨晕染过渡、朱红与赭石配色、古籍卷轴式排版——每一处视觉细节都在提醒你：这不是普通的导览 App，这是一本可以走进的紫禁城密档。

### 5. 秘辛地图与成就收藏
完成一处勘验，解锁一处秘辛。所有已探索的剧情收录于「秘辛收藏册」，支持随时回溯重读，也能切换不同视角重新体验同一景点。

## 技术架构

```
┌─────────────────┐     ┌─────────────────┐     ┌─────────────────┐
│   visitor-web   │────▶│   Express API   │────▶│   Prisma ORM    │
│  (React + Vite) │     │  (TypeScript)   │     │   (SQLite/...)  │
└─────────────────┘     └─────────────────┘     └─────────────────┘
```

- **前端**：React 18 + Vite + Tailwind CSS + React Router，移动端优先的 PWA 体验
- **后端**：Express + TypeScript + Prisma，RESTful API 设计
- **AI 集成**：LLM 叙事生成 + RAG 知识库问答 + ASR 语音识别 + TTS 语音合成
- **算法**：四因子加权评分 + 隐式反馈动态画像更新

## 核心页面流

1. **Start（勘验入口）** — 古籍翻页开场动画，扫码即入
2. **Interest（兴趣选择）** — 选择你偏好的叙事视角
3. **Explore（秘辛地图）** — 全景点位地图，AI 推荐下一处秘辛
4. **Navigate（寻路导航）** — 步行引导，距离与时间预估
5. **Narrative（剧情叙事）** — AI 生成专属故事 + 语音交互问答
6. **Complete（勘验完成）** — 行程回顾、收藏册解锁、成就分享卡
