# 叙境 — 领域词汇表

## 核心概念

**Visitor（游客）**
进入景区并使用叙境导览的最终用户。通过扫码进入，选择兴趣标签，跟随剧情路线游览。

**Spot（点位）**
景区内的具体位置：热门展品、冷门展区、商户店、入口、排队区。是剧情路线和打卡的基本单元。

**Narrative（剧情）**
AI 生成的个性化游览故事。不是固定讲解，而是根据游客兴趣和现场状态动态生成的任务化内容。

**Route（路线）**
游客实际走过的点位序列。系统在每一步根据策略决定下一个推荐点位。

**Check-in（打卡）**
游客到达目标点位的确认动作。MVP 支持二维码扫码和定位两种方式。

**Reward（奖励）**
任务完成后发放给游客的权益：商户优惠券、电子徽章、隐藏剧情、纪念卡。

**Campaign（活动）**
景区运营人员在后台配置的一次完整叙事导流活动，包含点位、剧情、目标、奖励的完整配置。

**Status（点位状态）**
点位的运营标记：正常、拥挤、暂停引导、重点引导。用于自适应路线决策。

**RAG Corpus（资料库）**
景区官方资料、展品介绍、历史故事等结构化知识库。AI 生成内容的约束来源，禁止编造无出处事实。

## 参与者

**Operator（运营人员）**
景区侧配置活动的人员。通过后台上传资料、配置点位和状态、查看数据复盘。

**Merchant（商户）**
景区内的文创店或餐饮等服务提供方。提供优惠券作为剧情奖励，通过扫码核销。

## 关键数据

**Interest Tag（兴趣标签）**
游客选择的偏好：历史、建筑、人物、亲子、悬疑、工艺。决定叙事风格和内容方向。

**Task（任务）**
剧情中的具体行动指令，通常引导游客前往下一个点位。例如"去钟表馆找第二条线索"。

**Insight（语义反馈）**
从游客提问和自由反馈中提取的经营建议，如"内容太浅""排队时间长""文创偏好"。
