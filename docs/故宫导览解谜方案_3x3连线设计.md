# 故宫导览解谜方案：3×3 连线设计

## 方案概览

| 角色 | 选择 | 理由 |
|------|------|------|
| **出发点（3个高压区）** | **钟表馆**（奉先殿）<br>**珍宝馆**（宁寿宫区）<br>**武英殿·陶瓷馆** | ①②是故宫最热的两个收费专馆，常年排队；③是"瓷母"等镇馆之宝所在地，重磅特展时人流极高 |
| **目的地（3个低谷区）** | **延禧宫**（东六宫深处）<br>**寿康宫**（外西路）<br>**慈宁宫**（外西路） | 均偏离中轴线，旅行团不经过；均有特色展览/原状陈列；且与出发点存在真实的历史文化关联 |

> **策略说明**：不需要每个宫都设。只挑选 3 个流量高压区作为"出发点"，3 个流量低谷区作为"引导目的地"。一共只需要维护 9 组碎片信息（即 3×3 的连线组合）。

---

## 第一行：从【钟表馆】出发

**核心叙事锚点**：中西合璧的机械奇技——清宫造办处与西洋贡钟的碰撞。

### 1.1 钟表馆 → 延禧宫

| 字段 | 内容 |
|------|------|
| **当前馆** | 钟表馆（人多） |
| **独特特征** | 铜镀金写字人钟的机械传动 |
| **跳转馆** | 延禧宫 |
| **利益** | 延禧宫"水晶宫工匠"御制印章 |
| **关联逻辑** | 写字人钟的齿轮传动与延禧宫灵沼轩（水晶宫）的西洋钢铁架构，同属清末"中西合璧"工法 |

**生成的文稿：**

> "这写字人钟的齿轮传动，其实并不是中国工匠的独创。你看它底座的纹路，与延禧宫当年的水殿遗存如出一辙。去延禧宫，那里的残图能揭开这段中西合璧的工法秘密。若你寻回了线索，可获得延禧宫御制金印一枚。"

---

### 1.2 钟表馆 → 寿康宫

| 字段 | 内容 |
|------|------|
| **当前馆** | 钟表馆（人多） |
| **独特特征** | 紫檀嵌珐琅重檐楼阁更钟（融合中国传统打更与西方机械） |
| **跳转馆** | 寿康宫 |
| **利益** | 《崇庆皇太后八旬万寿图》数字收藏卡 |
| **关联逻辑** | 乾隆年间造办处为崇庆皇太后八旬万寿特制的"自动贺寿钟"原型，与寿康宫原状陈列中的太后起居场景直接关联 |

**生成的文稿：**

> "这座更钟的机芯里藏着一个小秘密——乾隆年间，造办处为讨好一位老佛爷，专门做过一种'会自动跪下贺寿'的奇器。那位老佛爷，就住在寿康宫。去那里，在后殿的黄花梨大柜旁，找到那张藏在暗格里的贺寿钟图纸。"

|------|------|

## 第二行：从【珍宝馆】出发

**核心叙事锚点**：宫廷珍宝的温度与归属——冰冷的金银器需要"生活场景"来赋予意义。

### 2.1 珍宝馆 → 寿康宫

| 字段 | 内容 |
|------|------|
| **当前馆** | 珍宝馆（人多） |
| **独特特征** | 清宫后妃的珐琅工艺 |
| **跳转馆** | 寿康宫 |
| **利益** | 寿康宫"宫廷印记" |
| **关联逻辑** | 珍宝馆的金银细工 ↔ 寿康宫后妃手制珐琅小件的生活温度 |

**生成的文稿：**

> "这些金银细工在珍宝馆里显得过于冰冷，因为它们缺少了当年的生活温度。寿康宫里藏着当年后妃们手制的珐琅小件，那里才是这些珍宝原本的归宿。去寻回它们的温度吧，那是开启寿康宫宫廷印记的唯一线索。"

### 2.2 珍宝馆 → 延禧宫

| 字段 | 内容 |
|------|------|

## 第三行：从【武英殿·陶瓷馆】出发

**核心叙事锚点**：御窑瓷器的"主人密码"——每一件官窑背后都有一位具体的使用者。

### 3.1 陶瓷馆 → 延禧宫

| 字段 | 内容 |
|------|------|
| **当前馆** | 武英殿·陶瓷馆（人多） |
| **独特特征** | "瓷母"（各色釉彩大瓶）集十七种釉彩于一身 |
| **跳转馆** | 延禧宫 |
| **利益** | 延禧宫"釉彩拼图"数字藏品 |
| **关联逻辑** | 瓷母集十七种釉彩，但缺少西洋透明珐琅彩 ↔ 延禧宫藏有欧洲珐琅玻璃器，正是瓷母上"缺失的最后一笔" |

**生成的文稿：**

> "这只瓷母身上聚集了十七种釉彩，堪称中西工法的集大成者。但你可知道，有一种釉彩它学不来——那种只在西洋玻璃器上才有的透明珐琅。延禧宫，那座水晶宫里，藏着一批从欧洲来的珐琅玻璃器，正是瓷母上缺失的那最后一笔。去延禧宫，找到那件能让瓷母'完整'的西洋珐琅样。"

## 设计说明

### 1. 为什么选这3个出发点？

- **钟表馆**和**珍宝馆**是故宫唯二需要额外购票（各10元）的常设专馆。根据故宫数据，2018年仅专馆类参观就达524万人次，是铁打的高压区。
- **武英殿·陶瓷馆**虽不额外收费，但有"瓷母"（各色釉彩大瓶）等超级镇馆之宝，每当展出时人流极大，且陶瓷与后宫生活用瓷的关联天然适合讲故事。

### 2. 为什么选这3个目的地？

- **延禧宫**：位置偏东六宫深处，虽有《延禧攻略》知名度但实地游客远低于中轴线；其"水晶宫"（灵沼轩）是故宫唯一的西洋建筑遗存，视觉差异度极高。
- **寿康宫**：2015年才首次开放，位于外西路需特意绕行，团队游客极少；有220余件原状陈列文物，是"有内容可挖"的清净角落。
- **慈宁宫**：同样在外西路，是清初孝庄太后的居所，有雕塑馆展览；相比寿康宫知名度更低（多数游客只知寿康宫的"甄嬛"而不知慈宁宫的"孝庄"），分流效果极佳。

### 3. 路线分布合理性

- **东侧目的地**：延禧宫（承接从钟表馆、珍宝馆、陶瓷馆东线漫游的游客）
- **西侧目的地**：寿康宫、慈宁宫（形成外西路"太后双宫"的小环线，两宫相距仅200米左右，但各自有完全不同的历史叙事：孝庄 vs 崇庆皇太后）

### 4. 需要验证的事项

- [ ] 寿康宫原状陈列中是否有崇庆皇太后专属堂号的粉彩瓷器（如"乐寿堂"款等），建议与故宫展陈部确认具体展品。
- [ ] 慈宁宫雕塑馆的具体展陈是否有与孝庄太后直接相关的文物或说明牌。
- [ ] 武英殿当前是否展出郎窑红釉瓶及各色釉彩大瓶（重量级文物可能轮换展出）。

---

> **方案优势**：每个关联都建立在真实的历史工艺或人物关系上，不是生拉硬凑。玩家从人挤人的馆里接到线索后，会带着"解谜目标"主动走向那些清净的宫殿，起到了真实的客流引导作用。
