## 二、逐点位数据明细

### 1. 钟表馆（spot-clock）

**基本信息**
- 位置：奉先殿（紫禁城内廷东侧，东六宫以南）
- 始建：明永乐十八年（1420年）
- 清代重建：顺治十四年（1657年）
- 改为钟表馆：1925年故宫博物院成立后
- 馆藏规模：中外钟表二百余件
- 建筑形制：工字形大殿，黄琉璃瓦重檐庑殿顶，金龙和玺彩画

**核心展品**
| 展品名称 | 说明 |
|---|---|
| 铜镀金写字人钟 | 英国伦敦威廉森制造，通高2.31米，底层写字机械人可书写"八方向化，九土来王" |
| 硬木雕花楼式自鸣钟 | 故宫最大自鸣钟，高约6米、宽逾2米 |
| 铜壶滴漏 | 故宫唯一存世古代滴漏 |

**6个兴趣维度叙事**

| 兴趣 | 标题 | 核心史实摘要 |
|---|---|---|
| 历史 | 时间的帝国 | 1420年始建，顺治1657年重建，1925年改钟表馆；馆藏分广州/苏州/造办处国产与英法德日洋货 |
| 建筑 | 工字大殿的庄严 | 工字形布局，前殿九间、后殿五间，庑殿顶为最高等级，白石须弥座台基 |
| 人物 | 康熙与西洋钟表匠 | 康熙设自鸣钟处（隶属造办处），利类思、安文思、南怀仁等曾效力 |
| 亲子 | 寻找会写字的钟 | 写字人钟为英国使团贡品，硬木雕花楼式自鸣钟为故宫最大 |
| 悬疑 | 停走的铜壶滴漏 | [待核实] 铜壶滴漏"无故停走七日"及内部金属薄片为艺术加工，非故宫档案记录 |
| 工艺 | 珐琅、黄金与齿轮 | 齿轮精度0.01mm，掐丝珐琅以宝石粉末为釉，800度高温烧制 |

**待核实项**
- [ ] 铜壶滴漏停走七日及金属薄片：需确认是否为真实档案记录，或需改为纯虚构标注
- [ ] 馆藏钟表具体数量（二百余件）是否与最新官方数据一致

### 2. 珍宝馆（spot-treasure）

**基本信息**
- 位置：宁寿宫区（紫禁城内廷东北部）
- 营建：乾隆三十六至四十一年（1771—1776年）
- 占地面积：约5万平方米
- 原用途：乾隆皇帝为自己准备的太上皇宫
- 建筑形制：仿紫禁城中路格局，含皇极殿、宁寿宫、畅音阁、花园等

**核心展品**
| 展品名称 | 说明 |
|---|---|
| 金瓯永固杯 | 约500克黄金打造，镶嵌珍珠/宝石/红蓝宝石，元旦饮屠苏酒专用 |
| 点翠嵌珠后妃凤冠 | 明万历年间孝端显皇后所用，漆竹丝为胎，翠鸟羽毛点翠 |
| 田黄三连章 | 乾隆帝御用印章 |
| 象牙编织席 | 象牙劈成0.2毫米薄片编织，乾隆后禁止制作 |

**6个兴趣维度叙事**

| 兴趣 | 标题 | 核心史实摘要 |
|---|---|---|
| 历史 | 太上皇的宝库 | 1771—1776年乾隆建太上皇宫，未入住；后供皇太后/太妃居住，慈禧晚年居此 |
| 建筑 | 紫禁城的缩影 | 前朝后寝规制完整，含皇极殿、宁寿宫、养性殿、乐寿堂、畅音阁、花园等 |
| 人物 | 崇庆皇太后的寿辰 | 崇庆皇太后（1692—1777）八旬万寿为清代最盛大庆典之一 |
| 亲子 | 寻找金瓯永固杯 | 金瓯永固杯为皇帝元旦专用酒杯，镶嵌珍珠宝石 |
| 悬疑 | 失传的点翠技艺 | [待核实] 凤冠东珠"十二颗"vs"十颗"差异为艺术加工，非真实档案矛盾 |
| 工艺 | 金玉之工 | 宝石镶嵌误差<0.1mm，点翠需数百只翠鸟，象牙席已禁止 |

**待核实项**
- [ ] 凤冠东珠数量差异：需确认是否为剧情虚构，避免被误认为真实历史争议
- [ ] 金瓯永固杯重量（500克）与具体镶嵌物清单
- [ ] 点翠工艺所需翠鸟数量（数百只）为估算，需确认表述是否准确

### 3. 武英殿·陶瓷馆（spot-ceramic）

**基本信息**
- 位置：紫禁城外朝西路（与文华殿东西对称）
- 始建：明永乐十八年（1420年）
- 清代用途：康熙十九年（1680年）设修书处，刊刻《武英殿聚珍版丛书》
- 改为陶瓷馆：2008年
- 馆藏范围：新石器时代至清末陶瓷精品
- 建筑形制：黄琉璃瓦歇山顶，面阔五间，单翘单昂五踩斗栱

**核心展品**
| 展品名称 | 说明 |
|---|---|
| 汝窑天青釉器 | 宋代五大名窑之首，以玛瑙入釉 |
| 元青花 | 以苏麻离青料绘制，发色浓艳 |
| 成化斗彩鸡缸杯 | 全球仅存十余件真品 |
| 粉彩九桃天球瓶 | 雍正/乾隆官窑精品，唐英督陶时期 |
| 各种釉彩大瓶 | 俗称"瓷母"，集合多种釉彩工艺 |

**6个兴趣维度叙事**

| 兴趣 | 标题 | 核心史实摘要 |
|---|---|---|
| 历史 | 从斋宫到陶瓷圣殿 | 1420年始建，明末李自成在此即位，康熙设修书处，2008年改陶瓷馆 |
| 建筑 | 歇山顶与聚珍版 | 歇山顶五间，东西配殿耐寒/恒寿殿，后有敬思殿、主敬殿 |
| 人物 | 督陶官唐英 | 唐英（1682—1756）驻景德镇督陶二十余年，著《陶冶图说》 |
| 亲子 | 寻找陶瓷上的动物园 | 定窑游鱼、青花龙纹、成化鸡缸杯小鸡、粉彩仙鹤蝙蝠 |
| 悬疑 | 碎瓷片上的密码 | [待核实] "碎瓷片拼合鸡缸杯"为艺术加工，非真实发现 |

**待核实项**

- [ ] "瓷母"（各种釉彩大瓶）是否确为武英殿陶瓷馆藏

### 4. 延禧宫（spot-yanxi）

**基本信息**
- 位置：紫禁城东六宫之一
- 始建：明永乐十八年（1420年），初名长寿宫
- 改名：嘉靖十四年（1535年）改延禧宫
- 火灾：清光绪二十年（1894年）毁于火灾
- 改建：宣统元年（1909年）隆裕太后下令修建"水晶宫"（灵沼轩）
- 现状：紫禁城唯一西洋式烂尾楼，钢筋混凝土结构

**核心看点**
| 看点 | 说明 |
|---|---|
| 灵沼轩遗址 | 清末计划中的三层玻璃水殿，地下一层蓄养金鱼 |
| 西洋式结构 | 钢筋混凝土骨架，巴洛克风格装饰，钢材可能来自德国 |
| 停建原因 | 1911年辛亥革命爆发，工程被迫停建 |

**6个兴趣维度叙事**

| 兴趣 | 标题 | 核心史实摘要 |
|---|---|---|
| 历史 | 东六宫里的西洋梦 | 1420年始建，初名长寿宫，1535年改名，1894年火灾，1909年建水晶宫 |
| 建筑 | 紫禁城的唯一烂尾楼 | 汉白玉须弥座+钢筋混凝土+巴洛克装饰，1909年技术 |
| 人物 | 隆裕太后的心愿 | 隆裕太后（1868—1913）为慈禧侄女，垂帘听政，1913年病逝 |
| 亲子 | 如果水晶宫建成了 | 三层玻璃结构，地下养金鱼，透明地板 |
| 悬疑 | 地下室里的图纸 | [待核实] 2010年发现铜制圆筒及德文信件为艺术加工，非真实发现 |
| 工艺 | 玻璃、钢铁与汉白玉 | 汉白玉须弥座+竹节钢骨架+平板玻璃+进口瓷砖 |

**待核实项**
- [ ] 地下室发现铜制圆筒及德文图纸：纯虚构剧情，需确认是否已明确标注
- [ ] 钢材来源（德国）为推测，需确认是否有文献支持
- [ ] 灵沼轩具体设计层数（三层/四层争议）是否为真实历史讨论

### 5. 寿康宫（spot-shoukang）

**基本信息**
- 位置：紫禁城内廷外西路
- 始建：清乾隆元年（1736年）
- 建成：乾隆十四年（1749年）
- 用途：乾隆为生母崇庆皇太后建造的颐养之所
- 居住者：崇庆皇太后（42年）、孝和睿皇后、康慈皇太妃等
- 现状：原状陈列（2015年首次开放）

**核心看点**
| 看点 | 说明 |
|---|---|
| 崇庆皇太后原状陈列 | 2015年首次开放，展示太后真实生活场景 |
| 通体大炕 | 以黄土/白灰/沙子夯制，表面铺金砖 |
| 黄花梨雕龙大柜 | 故宫现存最大的黄花梨柜，2015年修缮时发现暗格珍宝 |
| 紫檀攒竹缂丝嵌玉屏风宝座 | 乾隆为母后八旬万寿特制的寿礼 |

**6个兴趣维度叙事**

| 兴趣 | 标题 | 核心史实摘要 |
|---|---|---|
| 历史 | 崇庆皇太后的福寿全归 | 1736年始建，1749年建成；崇庆皇太后享年86岁，为清代最长寿皇太后 |
| 建筑 | 为一位母亲设计的家 | 两进院落，五间歇山顶，门槛较低、地面平整、采光充足 |
| 人物 | 甄嬛的原型 | 崇庆皇太后钮祜禄氏（甄嬛原型）十三岁入雍亲王府，以母凭子贵晋升 |

| 工艺 | 紫檀、玉雕与苏绣 | 榫卯结构家具、紫檀嵌玉屏风、和田玉、苏绣（每平方厘米超三十针） |

**待核实项**

- [ ] 崇庆皇太后是否确实享年86岁（或85岁，按虚岁/周岁计算差异）

## 五、数据库初始化命令

```bash
cd server

```

- 1 个 Campaign（故宫密档·寻踪）
- 6 个 Spot（3 hot + 3 cold）
- 36 条 NarrativeTemplate（6点位 × 6兴趣标签）
