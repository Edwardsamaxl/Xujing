📱 寻迹 AI (TraceAI) 交互逻辑全流程
第一阶段：系统初始化 (静默闭环)
扫码/进入 (确认出发点)： 页面自动识别地理位置 Location_ID。
身份生成 (确认ID)： 系统在后台静默分配 Session_ID，并将游客匹配至预设的三个阵营之一（如：历史学者、剧情侦探、文创达人）。
风格选择 (选择讲解风格)： 弹出极简卡片：[严谨考据风] / [悬疑探案风] / [轻松幽默风]。
第二阶段：核心交互流 (数据探针)
剧情开启 (AI 动态叙事)： 基于 Location_ID + 风格设定，AI 实时生成 15 秒语音讲解。
交互探针 (用户提问/剧情决策)： 此时不设复杂的输入框，只提供三个带有行为标签的选项（例如：[调查工艺细节] -> 触发“考据型”标签；[审问NPC/寻找线索] -> 触发“剧情型”标签）。
任务确认 (确认游玩)： 点击“确定启程”，系统即刻将该用户纳入“空间流量调度”的计算池。
第三阶段：动态引流流转 (核心治理逻辑)
实时路径引导 (剧情提示下一步)： * 若当前区拥堵：AI 引导去冷门区（如：“这里线索已断，去东侧副展厅看看”）。
若当前区平稳：AI 引导进行深度互动（如：“发现关键道具，前往下一展柜”）。
平台轨迹同步 (地图可视化)： 游客手机端显示动态轨迹；评委看的 B 端后台大屏同步出现光点移动轨迹。
循环触发： 到达下一景点 -> 触发剧情交互 -> 引导至下一点位。
第四阶段：闭环结算与激励 (留存与变现)
勋章结算 (获得勋章)： 剧情终点，系统根据全程表现（任务完成度、探索覆盖率）赋予数字勋章。
路径管理 (底部 Button - 切换路线)： 底部固定 Button 允许用户在[深度考古线]、[全域打卡线]、[限时速览线]之间自由切换。
总结页 (结语+勋章墙)： 勋章墙展示已解锁的数字资产，并附带针对该游客偏好的个性化“文创购买链接”。
