# 叙境打卡勋章设计方案

本文档定义 6 个故宫试点地点的打卡勋章。所有勋章统一为“立体 3D 掐丝珐琅金印”，每枚根据地点代表性文物、建筑或人物记忆建立中心纹样，并在多宝格中以 PBR 金属与手机姿态光影呈现。

## 1. 统一母版

### 1.1 形态

- 基础轮廓：圆角八边形金印，外轮廓厚 6-8mm，正面微微鼓起，背面有浅浮雕印钮结构。
- 视觉结构：三层同心结构。
  - 外圈：高浮雕鎏金边框，刻“叙境密档”极细篆意暗纹。
  - 中圈：金丝云雷纹环绕，云纹走柔线，雷纹走折线，形成古意与数字线路感的融合。
  - 中央：地点专属抽象纹样，采用掐丝金线分区，内部填透明或半透明珐琅。
- 工艺语言：不是普通徽章，不做扁平图标。必须有厚度、倒角、金丝凸起、珐琅凹面、釉色深浅和边缘高光。
- 尺寸建议：多宝格缩略态 64px，解锁升起态 180-220px，3D 源模型建议以 1:1 单位制作。

### 1.2 材质

- 金属：暖金，接近故宫鎏金器物，不做亮黄塑料金。
  - baseColor: `#D7A84A`
  - metallic: `1.0`
  - roughness: `0.22-0.32`
  - clearcoat: `0.45`
  - bevel highlight: `#FFF0B8`
- 珐琅：半透明釉面，颜色来自地点特征。中央釉面略低于金丝 1-2mm，产生真实掐丝层次。
  - roughness: `0.12-0.2`
  - transmission: `0.15-0.25`
  - clearcoat: `0.7`
- 暗部：凹槽使用深茶金与朱砂暗影，不用纯黑。
  - shadowGold: `#5F4320`
  - cinnabarDark: `#7D2524`
- 光效：多宝格中使用一条斜向软高光扫过，随 deviceorientation 轻微移动，幅度控制在 4-8deg。

### 1.3 解锁动效

完成“清灰 + 补色 + 落印”后触发：

1. 清灰完成：屏幕中央浮起细尘，尘粒向外散开，露出暗金线稿。
2. 补色完成：珐琅釉色从中心向外漫开，像颜料被金丝边界自然截住。
3. 落印完成：金印下压 6px，出现一圈朱砂印痕，然后金印回弹升起。
4. 获得勋章：勋章从屏幕中心缓缓升起，旋转 `Y: -8deg -> 8deg`，金丝闪过一条流光。
5. 吸入多宝格：勋章缩小并沿贝塞尔曲线飞入对应格子，格子边缘泛起一圈淡金流光，持续 900ms 后收束。

缓动建议：升起 `800ms cubic-bezier(.2,.8,.2,1)`，吸入 `650ms cubic-bezier(.22,.61,.36,1)`。避免弹跳感太强，整体要像贵重器物被收入匣中。

## 2. 地点勋章设计

### 2.1 钟表馆：天机齿轮金印

- 地点关键词：机械钟表、西洋贡品、写字人钟、铜壶滴漏、宫廷时间。
- 中央纹样：抽象“写字人钟齿轮纹”。中央是一枚同心齿轮，齿轮内嵌一支极简毛笔笔尖，外侧用滴漏水珠形小珐琅点连接。
- 纹样逻辑：齿轮代表西洋机械，笔尖代表写字人钟，水滴代表铜壶滴漏，把“时间”“机械”“书写”合为一枚印。
- 主珐琅色：深孔雀蓝 `#0F5E6A`，表现钟表馆的精密与冷光。
- 辅助珐琅色：象牙白 `#F1E5C8`，用于笔尖与钟盘刻度。
- 金丝处理：齿轮齿尖必须金线凸起，钟盘刻度用极细金丝，不能画成普通线稿。
- 外圈云雷纹：云纹中穿插微型罗马刻度与水滴点，近看有“机械时标”的隐藏细节。
- 解锁文案：`天机齿轮金印已归匣`
- 资产提示词：

```text

```

### 2.2 珍宝馆：金瓯龙纹金印

- 地点关键词：宁寿宫区、皇家珍宝、金瓯永固杯、点翠凤冠、田黄印章、宝石镶嵌。
- 中央纹样：抽象“金瓯龙纹”。中央为一只俯视角宝杯轮廓，杯身化为盘龙曲线，龙身不写实，只保留龙脊、龙爪和卷云形龙尾。
- 纹样逻辑：金瓯永固杯代表帝国永固的祈愿，龙纹代表皇家珍宝等级，宝石点缀回应珍宝馆的璀璨。
- 主珐琅色：帝王绛红 `#8F1F2D`。
- 辅助珐琅色：祖母绿 `#0E6B53`，蓝宝石 `#183E8C`，仅作为小面积宝石点。
- 金丝处理：龙脊用连续凸金线，宝杯口沿做双层金丝，杯足处增加细密錾刻感。
- 外圈云雷纹：云雷纹更密，四角各嵌一颗微型珐琅宝石，营造“多宝”感。
- 解锁文案：`金瓯龙纹金印已归匣`
- 资产提示词：

```text
luxury 3D cloisonne enamel golden seal medal, rounded octagonal imperial seal, raised warm gold filigree, central abstract dragon wrapped around a ceremonial golden cup inspired by the Jin'ou Yonggu cup, ruby red translucent enamel, emerald and sapphire enamel gem dots, dense cloud-thunder border, high-end PBR gilded metal, polished bevels, sparkling but tasteful, museum artifact quality, no flat logo, no text
```

### 2.3 武英殿·陶瓷馆：万釉瓷轮金印

- 地点关键词：陶瓷馆、各色釉彩大瓶、成化斗彩鸡缸杯、汝官哥钧定、土与火。
- 中央纹样：抽象“万釉瓷轮”。中央为一只俯视瓷瓶口，向外扩散 6 片釉色花瓣，每片代表一种釉彩。底部藏一枚极简鸡纹，呼应斗彩鸡缸杯。
- 纹样逻辑：瓷器以“火中成色”为核心，勋章用多釉分区体现陶瓷文明的谱系感。
- 主珐琅色：天青釉 `#A9C8C0`。
- 辅助珐琅色：郎窑红 `#9D2525`，霁蓝 `#1E4B78`，米黄釉 `#D8BE82`，斗彩绿 `#4D8B57`。
- 金丝处理：每片釉色花瓣之间用金丝分隔，瓶口内圈做细金环，鸡纹只保留冠、尾、足三处符号。
- 外圈云雷纹：云雷纹间加入窑火火焰形转折，像云纹又像火焰。
- 解锁文案：`万釉瓷轮金印已归匣`
- 资产提示词：

```text
3D cloisonne enamel golden seal medal, octagonal palace seal with raised gilded filigree, central abstract porcelain vase mouth seen from above, six enamel glaze petals, celadon blue green main enamel, oxblood red, cobalt blue, warm yellow and doucai green accents, tiny abstract rooster motif hidden at the bottom, cloud thunder border blended with kiln flame shapes, premium PBR metal, glossy enamel, museum luxury, no text, no flat icon
```

### 2.4 延禧宫：灵沼羽光金印

- 地点关键词：东六宫、灵沼轩、水晶宫遗梦、西洋建筑遗迹、未完成的宫殿。
- 中央纹样：抽象“鸟纹与水晶窗”。中央是一只对称展开的金丝鸟纹，鸟翼由几何玻璃窗格组成，腹部是一滴灵沼水纹。
- 纹样逻辑：鸟纹回应用户认知中的延禧宫记忆，玻璃窗格回应灵沼轩“水晶宫”想象，水纹回应“灵沼”。
- 主珐琅色：水晶青 `#7FB7B1`，透明度略高。
- 辅助珐琅色：月白 `#E7E3D6`，浅金绿 `#BFC66B`。
- 金丝处理：鸟翼窗格必须是凸起金丝几何线，外缘略像巴洛克卷草，但保持东方纹样秩序。
- 外圈云雷纹：云雷纹中嵌入菱形玻璃格，形成“宫墙古纹 + 西洋窗棂”的混合。
- 解锁文案：`灵沼羽光金印已归匣`
- 资产提示词：

```text

```

### 2.5 寿康宫：福寿宝座金印

- 地点关键词：崇庆皇太后、颐养居所、原状陈列、紫檀家具、福寿安康。
- 中央纹样：抽象“宝座福寿纹”。中央是宝座背屏轮廓，背屏内用金丝组成寿字变体，左右各一枚如意云头，底部以紫檀木纹式珐琅暗纹托起。
- 纹样逻辑：寿康宫不是炫耀型珍宝，而是“太后生活空间”的温厚尊贵，所以纹样更稳、更圆润。
- 主珐琅色：紫檀褐 `#5B2F2A`。
- 辅助珐琅色：暖杏金 `#D9A75F`，玉白 `#E8DDC9`。
- 金丝处理：寿字不直接写成文字，要做成卷草几何化的“福寿结”，避免像廉价纪念币。
- 外圈云雷纹：云雷纹走宽缓节奏，四方加入如意头，整体比珍宝馆更内敛。
- 解锁文案：`福寿宝座金印已归匣`
- 资产提示词：

```text
premium 3D cloisonne enamel golden seal medal, rounded octagonal palace seal, warm gilded raised filigree, central abstract throne back motif with a stylized longevity knot, ruyi cloud shapes on both sides, dark red sandalwood brown translucent enamel, warm apricot gold and jade white accents, elegant cloud thunder border, dignified and calm imperial mother palace feeling, PBR metal, glossy enamel, beveled artifact, no readable text, no flat icon
```

### 2.6 慈宁宫：铜鹤佛光金印

- 地点关键词：皇太后正宫、慈宁宫花园、铜鹤铜龟、雕塑馆、礼佛与造像。
- 中央纹样：抽象“铜鹤佛光纹”。中央为一只立鹤剪影，鹤身由佛像背光的莲瓣圆光包裹，鹤足下是龟甲几何纹。
- 纹样逻辑：铜鹤与铜龟是殿前长寿象征，雕塑馆与礼佛记忆通过莲瓣背光表达，兼具宫廷与宗教艺术气质。
- 主珐琅色：古铜绿 `#3F6F63`。
- 辅助珐琅色：莲瓣白 `#E9DFCB`，暗金棕 `#7A5A2D`。
- 金丝处理：鹤颈与莲瓣背光必须用一笔连贯金丝，龟甲纹压在底部，不能抢中央鹤纹。
- 外圈云雷纹：云雷纹外叠一圈莲瓣短纹，形成“宫殿边框 + 造像背光”的识别。
- 解锁文案：`铜鹤佛光金印已归匣`
- 资产提示词：

## 3. 多宝格呈现规范

### 3.1 格子状态

- 未获得：只显示暗金线稿，珐琅区为低饱和灰釉，金属 roughness 提高到 `0.55`，像蒙尘。
- 进行中：中心纹样有一处被擦亮，格子右上角出现小金点。
- 已获得：完整 PBR 材质，格子底部出现地点短名，边缘保留淡淡流光。

### 3.2 手机晃动光影

- 使用 `deviceorientation` 或指针移动模拟旋转。
- 旋转范围：`rotateX: -6deg to 6deg`，`rotateY: -8deg to 8deg`。
- 高光层：用一条 `linear-gradient(115deg, transparent 35%, rgba(255,245,190,.55) 50%, transparent 65%)` 叠加在勋章表面，位置随姿态移动。
- 金丝和珐琅分层：金丝高光强，珐琅高光软。前端静态实现时至少拆成 `metal layer`、`enamel layer`、`shine layer` 三层。

### 3.3 声音与触感

- 清灰：轻微砂粒声，音量低。
- 补色：柔和水滴或釉面铺展声。
- 落印：短促低频“咚”，同时触发一次轻震。
- 吸入多宝格：极轻金属合扣声，格子流光收束。

## 4. 前端实现建议

MVP 可先用 2.5D 分层 PNG/WebP 实现，后续替换为 GLB 或 shader。

### 4.1 资产分层

每枚勋章输出：

- `base-metal.webp`：金属底座和外圈。
- `enamel.webp`：珐琅色块，透明背景。
- `filigree.webp`：金丝掐丝线，透明背景。
- `shadow.webp`：内部凹槽阴影。
- `normal.png`：可选，用于 WebGL/PBR。
- `poster.webp`：无动效兜底图。

### 4.2 数据字段建议

```ts
export interface BadgeMedalDesign {
  spotId: string
  name: string
  unlockText: string
  motif: string
  enamelPalette: string[]
  metalPalette: string[]
  assetPrompt: string
}
```

### 4.3 视觉优先级

1. 先保证每枚一眼可区分：中心纹样和主珐琅色必须不同。
2. 再保证成套感：统一八边金印、统一云雷纹、统一材质光效。
3. 最后做隐藏细节：近看才发现齿轮刻度、玻璃窗格、斗彩鸡纹、龟甲纹等。
